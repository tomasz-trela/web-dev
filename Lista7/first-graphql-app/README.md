# first-graphql-app

## How to start the project

1. **Install dependencies**

   In the `first-graphql-app` directory, run:
   
   ```sh
   npm install
   ```

2. **Start the server**

   ```sh
   npm start
   ```
   or
   ```sh
   node src/index.js
   ```

3. **Access GraphQL Playground**

   Open your browser at [http://localhost:4000/graphql](http://localhost:4000/graphql)

---

### Notes
- The database (`todos.db`) is created and seeded automatically on first run.
- Requires Node.js (v16  newer recommended).
